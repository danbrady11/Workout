# MWF Training

A personal workout tracker for a Push/Pull/Legs split at EOS Gym.

## Features
- Track sets and weights for each exercise
- See previous session's weight as a reference
- Calendar view with workout history
- Log gym classes, hikes, and other activities with notes

## Deploy

1. Push to GitHub
2. Import repo in [Vercel](https://vercel.com)
3. Deploy — no build step needed, it's a static site

## Data

All data is stored in your browser's `localStorage`. Nothing is sent to any server.
